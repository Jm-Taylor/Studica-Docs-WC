Sensors
=======

The VMX Robotics Controller boasts an impressive capacity to interface with a broad spectrum of sensors. Nevertheless, the focus of this documentation is selectively directed toward a subset of these sensors currently integrated into the Studica Robotics platform. If you wish to include a sensor that is not mentioned, please refer to the HAL section for guidance on how to do it on your own.
